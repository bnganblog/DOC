# SW基础功能介绍

SolidWorks（简称SW）是达索系统（Dassault Systèmes）旗下的一款三维机械设计软件，广泛应用于机械设计、工业设计、工程分析等领域。本文将介绍SolidWorks的核心基础功能，帮助初学者快速上手。

## 草图绘制

草图是SolidWorks建模的基础，所有三维特征都从二维草图开始。

### 草图实体工具

- **直线**：绘制直线段，是构建草图轮廓的基本元素
- **矩形**：提供边角矩形、中心矩形、3点边角矩形等多种绘制方式
- **圆**：支持圆心和半径画圆、周边圆两种方式
- **圆弧**：提供圆心/起/终点画弧、切线弧、3点圆弧三种方式
- **样条曲线**：绘制自由曲线，适合复杂曲面轮廓
- **多边形**：绘制正多边形，可设置边数和内切圆/外接圆尺寸
- **椭圆**：绘制椭圆和部分椭圆

### 草图约束

约束是保证草图几何关系和尺寸精度的关键：

- **几何关系**：包括重合、水平、竖直、相切、同心、共线、对称、相等、平行、垂直等
- **尺寸标注**：通过标注尺寸精确控制草图元素的大小和位置
- **自动约束**：绘制时自动推断并添加几何关系
- **完全定义**：草图应处于完全定义状态（黑色），避免欠定义（蓝色）或过定义（红色）

## 特征建模

特征是SolidWorks三维建模的核心，通过在草图基础上添加特征来构建三维实体。

### 基础特征

| 特征 | 说明 |
|------|------|
| 拉伸凸台/基体 | 将草图沿垂直方向拉伸生成实体，可设置深度、方向和拔模角 |
| 旋转凸台/基体 | 将草图绕轴线旋转生成回转体，适合轴类、盘类零件 |
| 扫描 | 将轮廓沿路径移动生成实体，适合管道、弹簧等 |
| 放样 | 在多个轮廓之间过渡生成实体，适合复杂形状 |
| 边界凸台/基体 | 类似放样，但对曲面质量控制更精细 |

### 切除特征

切除特征用于从已有实体中去除材料：

- **拉伸切除**：从实体中切除拉伸区域
- **旋转切除**：绕轴旋转切除材料
- **扫描切除**：沿路径扫描切除
- **放样切除**：在轮廓间过渡切除
- **边界切除**：边界方式切除

### 工程特征

- **圆角**：对棱边进行圆角过渡，支持等半径、变半径、面圆角等方式
- **倒角**：对棱边进行倒角处理，支持角度距离和距离距离两种定义方式
- **抽壳**：将实体挖空形成薄壁件，可指定不同壁厚
- **筋**：在开环草图上生成加强筋
- **拔模**：为面添加拔模斜度，便于模具脱模
- **孔**：创建各种类型的孔特征，支持简单直孔和异型孔向导
- **阵列**：线性阵列、圆周阵列、曲线驱动的阵列、填充阵列等
- **镜像**：将特征关于基准面镜像复制

## 装配体

装配体功能用于将多个零件组合在一起，模拟真实产品的组装关系。

### 配合类型

- **标准配合**：重合、平行、垂直、相切、同轴心、锁定、距离、角度
- **高级配合**：对称、宽度、路径配合、线性/线性耦合
- **机械配合**：凸轮、齿轮、铰链、齿条小齿轮、螺旋、万向节

### 装配体操作

- **插入零部件**：将零件添加到装配体中
- **移动/旋转零部件**：调整零部件位置
- **配合**：定义零部件之间的几何关系
- **干涉检查**：检测零部件之间的干涉
- **爆炸视图**：创建装配体的分解视图
- **装配体特征**：在装配体级别创建切除等特征

## 工程图

工程图是将三维模型转化为二维制造图纸的功能模块。

### 工程图类型

- **标准三视图**：前视、上视、右视三个基本视图
- **模型视图**：从任意方向投影生成视图
- **投影视图**：从已有视图投影生成辅助视图
- **辅助视图**：垂直于边线投影的视图
- **剖面视图**：用剖切面切开模型显示内部结构
- **局部视图**：放大显示局部细节
- **断开的剖视图**：局部剖切显示内部结构
- **断裂视图**：缩短较长零件的显示

### 标注功能

- **尺寸标注**：自动或手动标注尺寸
- **表面粗糙度**：标注表面加工要求
- **形位公差**：标注形状和位置公差
- **焊接符号**：标注焊接要求
- **基准特征**：标注基准面和基准轴
- **注释**：添加文字说明
- **零件序号**：在装配图中标注零件编号
- **材料明细表（BOM）**：自动生成零件清单

## 曲面建模

曲面建模用于创建复杂形状，是高级造型的重要工具。

### 曲面功能

- **拉伸曲面**：将草图拉伸为曲面
- **旋转曲面**：将草图旋转为曲面
- **扫描曲面**：沿路径扫描生成曲面
- **放样曲面**：在轮廓间过渡生成曲面
- **边界曲面**：精确控制生成高质量曲面
- **填充曲面**：修补曲面缺口
- **平面区域**：由封闭轮廓生成平面
- **等距曲面**：偏移生成等距曲面
- **延伸曲面**：延伸曲面边界
- **剪裁曲面**：用曲面或草图剪裁曲面
- **缝合曲面**：将多个曲面缝合为整体
- **加厚**：将曲面加厚为实体

## 钣金设计

SolidWorks提供专门的钣金设计工具：

- **基体法兰**：创建钣金基础特征
- **边线法兰**：在边线上添加法兰
- **斜接法兰**：创建斜接的法兰特征
- **褶边**：在边线上创建褶边
- **绘制的折弯**：在草图线上折弯
- **转折**：创建Z形折弯
- **闭合角**：闭合钣金角落
- **展开/折叠**：展开或折叠折弯
- **成形工具**：使用成形工具创建冲压特征

## 焊件设计

焊件功能用于设计焊接结构：

- **结构构件**：从标准型材库中选择截面生成构件
- **剪裁/延伸**：剪裁构件端部
- **顶端盖**：封闭构件端部
- **角撑板**：添加角撑板加强
- **圆角焊缝**：添加焊缝符号和焊缝

## 总结

SolidWorks作为一款功能强大的三维设计软件，涵盖了从草图绘制、特征建模、装配体设计到工程图出图的全流程。掌握以上基础功能后，可以完成大多数机械零件和产品的设计工作。建议初学者从草图和基础特征入手，逐步学习装配和工程图，最终掌握曲面、钣金等高级功能。
