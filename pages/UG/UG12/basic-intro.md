# UG12基础功能介绍

Siemens NX（原名UG/Unigraphics）是西门子旗下的一款高端CAD/CAM/CAE一体化软件，广泛应用于航空航天、汽车制造、模具设计、机械加工等领域。本文将介绍NX12的核心基础功能，帮助初学者快速入门。

## 界面与基本操作

### 工作界面

NX12的工作界面由以下主要区域组成：

- **菜单栏**：包含文件、编辑、视图、插入、格式、工具等菜单
- **工具条**：提供常用功能的快捷按钮，可自定义显示
- **资源条**：集成了部件导航器、装配导航器、特征回放等面板
- **图形窗口**：三维模型的显示和操作区域
- **提示行/状态行**：显示操作提示和当前状态信息
- **快捷键**：支持丰富的键盘快捷操作

### 鼠标操作

| 操作 | 功能 |
|------|------|
| 左键 | 选择对象 |
| 中键（滚轮） | 旋转视图 / 确认操作 |
| 右键 | 弹出快捷菜单 |
| 中键滚轮 | 缩放视图 |
| Shift+中键 | 平移视图 |

### 视图操作

- **适合窗口**（Ctrl+F）：将模型调整到适合窗口大小
- **旋转**：围绕模型中心旋转视角
- **平移**：移动视图位置
- **缩放**：放大或缩小视图
- **正三轴测图**（Home键）：切换到标准等轴测视图
- **正视于**：正对选定平面查看

## 草图绘制

草图是NX建模的基础，所有三维特征几乎都从草图开始。

### 草图创建

- **在平面上**：选择已有平面或基准面作为草图平面
- **基于路径**：沿曲线路径创建草图
- **在坐标系上**：选择坐标系平面创建草图

### 草图曲线工具

- **轮廓**：连续绘制直线和圆弧
- **直线**：绘制直线段
- **圆弧**：支持三点圆弧、圆心和端点圆弧
- **圆**：支持圆心和半径、三点画圆
- **矩形**：多种矩形绘制方式
- **样条**：绘制通过点或极点的样条曲线
- **椭圆**：绘制椭圆
- **倒斜角**：对角点进行倒角
- **圆角**：对角点进行圆角过渡
- **艺术样条**：绘制自由形状的样条曲线

### 草图约束

- **尺寸约束**：标注长度、角度、直径等尺寸
- **几何约束**：
  - 重合、共线、同心
  - 水平、竖直
  - 平行、垂直
  - 相切、等长、等半径
  - 固定、点在曲线上
- **自动约束**：系统自动推断并添加约束
- **转换至/自参考对象**：将曲线转换为参考线

## 特征建模

### 基础成形特征

| 特征 | 说明 |
|------|------|
| 拉伸 | 将草图沿方向拉伸生成实体，支持对称拉伸、拔模等 |
| 旋转 | 将草图绕轴线旋转生成回转体 |
| 沿引导线扫掠 | 将截面沿引导线扫掠生成实体 |
| 管道 | 沿中心线创建管状实体 |
| 变化的扫掠 | 截面沿路径变化扫掠 |

### 高级成形特征

- **扫掠**：沿一条或多条引导线扫掠截面
- **变化的扫掠**：截面沿路径规律变化
- **桥接**：在两个面之间创建过渡面
- **N边曲面**：由多条边围合创建曲面
- **规律延伸**：按规律延伸曲面

### 设计特征

- **孔**：创建常规孔、钻形孔、螺钉间隙孔、螺纹孔等
- **凸台**：在平面上添加圆柱形凸起
- **腔体**：在实体上创建凹腔
- **垫块**：在平面上添加矩形或常规垫块
- **键槽**：创建直角坐标键槽
- **沟槽**：在圆柱面上创建环形沟槽
- **三角形加强筋**：创建加强筋特征
- **螺纹**：在圆柱面上创建螺纹

### 细节特征

- **边倒圆**：对棱边进行圆角过渡，支持可变半径、拐角回拨等
- **面倒圆**：对两组面之间进行圆角过渡
- **软倒圆**：创建横截面非圆的倒圆
- **倒斜角**：对棱边进行倒角
- **拔模**：为面添加拔模斜度
- **体拔模**：对实体进行拔模

### 关联复制

- **阵列特征**：
  - 线性阵列
  - 圆形阵列
  - 多边形阵列
  - 螺旋式阵列
  - 沿路径阵列
  - 常规阵列（点集）
- **镜像特征**：将特征关于基准面镜像
- **镜像体**：将整个体镜像
- **抽取几何特征**：从实体抽取面、曲线等
- **复合曲线**：复制曲线
- **实例几何体**：创建几何体的关联副本

## 曲面建模

NX的曲面建模功能非常强大，是工业设计的核心工具。

### 曲面创建

- **通过点**：通过点阵创建曲面
- **从极点**：通过控制极点创建曲面
- **从点云**：通过大量点数据拟合曲面
- **直纹面**：在两条截面线之间创建直纹曲面
- **通过曲线组**：通过多条截面线创建曲面
- **通过曲线网格**：通过主曲线和交叉曲线创建曲面
- **扫掠曲面**：沿引导线扫掠截面创建曲面
- **截面曲面**：创建二次截面曲面
- **N边曲面**：由边界曲线围合创建曲面
- **桥接曲面**：在两个曲面之间创建过渡曲面
- **延伸曲面**：延伸已有曲面边界
- **规律延伸**：按规律延伸曲面
- **偏置曲面**：偏移创建等距曲面
- **修剪片体**：用曲线或面修剪曲面
- **缝合**：将多个曲面缝合为整体
- **加厚**：将曲面加厚为实体

### 曲面编辑

- **移动定义点**：修改曲面上的定义点
- **移动极点**：修改曲面的控制极点
- **扩大**：扩大或缩小曲面
- **等参数修剪/分割**：沿参数方向修剪或分割曲面
- **更改阶次**：修改曲面的阶次
- **更改刚度**：修改曲面的刚度
- **法向反向**：反转曲面法线方向

## 装配建模

### 装配方法

- **自底向上**：先设计零件，再组装成装配体
- **自顶向下**：在装配环境中设计零件

### 装配约束

- **接触对齐**：使面贴合或对齐
- **同心**：使圆柱面同轴
- **距离**：控制对象之间的距离
- **固定**：固定零部件位置
- **平行**：使方向平行
- **垂直**：使方向垂直
- **对齐/锁定**：对齐并锁定方向
- **胶合**：将部件粘合在一起
- **中心**：使对象居中
- **角度**：控制角度关系

### 装配功能

- **WAVE几何链接器**：在装配中关联链接几何数据
- **干涉检查**：检测部件之间的干涉
- **爆炸视图**：创建装配分解视图
- **装配序列**：定义装配和拆卸顺序
- **装配切割**：在装配级别创建切割特征
- **部件阵列**：在装配中阵列部件
- **镜像装配**：镜像整个装配

## 工程制图

### 视图创建

- **基本视图**：从模型创建主视图
- **投影视图**：从已有视图投影生成辅助视图
- **局部放大图**：放大显示局部细节
- **剖视图**：
  - 全剖视图
  - 半剖视图
  - 旋转剖视图
  - 局部剖视图
  - 阶梯剖视图
- **断开视图**：缩短较长零件的显示

### 标注功能

- **尺寸标注**：自动/手动标注各类尺寸
- **表面粗糙度**：标注表面加工要求
- **形位公差**：标注形状和位置公差
- **基准特征符号**：标注基准
- **焊接符号**：标注焊接要求
- **注释**：添加文字说明
- **零件明细表**：自动生成BOM表
- **自动气泡标注**：自动编号标注

## 同步建模

同步建模是NX的特色功能，可以在无参数模型上进行修改：

- **移动面**：直接移动面改变模型形状
- **拉出面**：从面拉出区域
- **偏置区域**：偏移一组面
- **替换面**：用新面替换旧面
- **删除面**：删除面并自动修补
- **调整面大小**：修改圆柱面直径
- **调整圆角大小**：修改圆角半径
- **调整倒斜角大小**：修改倒角尺寸
- **共面**：使面共面
- **共轴**：使面共轴
- **对称**：使面关于基准面对称
- **编辑横截面**：通过截面修改模型

## 加工模块（CAM）

NX的加工模块是其核心优势之一：

### 加工类型

- **平面铣**：2D轮廓加工
- **型腔铣**：3D粗加工
- **固定轴曲面轮廓铣**：3D精加工
- **可变轴曲面轮廓铣**：五轴加工
- **钻孔**：各类孔加工
- **车削**：车床加工
- **线切割**：线切割加工

### 加工操作流程

1. 创建加工几何体（毛坯、部件、检查体）
2. 创建刀具
3. 创建加工方法
4. 创建操作并设置参数
5. 生成刀轨
6. 刀轨仿真验证
7. 后处理输出NC代码

## 总结

NX12作为一款功能全面的CAD/CAM/CAE软件，涵盖了从概念设计、详细设计、工程制图到加工制造的完整流程。其强大的曲面建模能力、独特的同步建模技术和完善的加工模块使其在高端制造领域占据重要地位。建议初学者从草图和基础特征入手，逐步掌握装配、工程图和曲面建模，最终学习加工编程等高级功能。
